🤖 AI-Powered Employee Management SystemA full-stack, 
containerized application featuring a Modern React UI, a Spring Boot Backend, and a MySQL Database, 
all integrated with Large Language Models (LLMs) for natural language data querying.

“I built an AI-powered employee assistant system using Spring Boot and React. The system allows employees to query company-related information. I designed REST APIs, managed data using JPA, and integrated an AI service to generate intelligent responses. I also implemented proper exception handling and modular architecture.”


🌟 Key FeaturesModern Glassmorphism UI: A sleek, responsive dashboard built with React and Tailwind CSS.AI Assistant: Natural language interface allowing users to "talk" to their database (e.g., "How many leaves are in annual?").
Full CRUD Operations: Seamlessly create, read, update, and delete employee records.
Dockerized Stack: Single-command deployment for the entire infrastructure.
